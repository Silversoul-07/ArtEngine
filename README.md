# ArtEngine

This is the backend of **Artverse**, powered by FastAPI with PostgreSQL and MilvusDB.

## Installation

```bash
git clone https://github.com/Silversoul-07/ArtEngine.git
cd ArtEngine
pip install -r requirements.txt
```

## Usage

```bash
uvicorn ArtEngine.main:app --reload
```

## Technologies

- FastAPI
- Python
- PostgreSQL
- MilvusDB
